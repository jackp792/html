Tags

  1. This is a <'link'> tag. The <'link'> tag defines the relationship between the current document and an external resource. The <'link'> tag is most often used to link to external style sheets or to add a favicon to your website.

  2. This is a <'title'> tag. The <'title'> tag is an HTML code tag that allows you to give a web page a title. This title can be found in the browser title bar, as well as in the search engine results pages.

  3. This is a <'body'> tag. The <'body'> tag defines the document's body. The <'body'> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

  4. This is a <'h1'> tag. The <'h1'> tag defines the most important heading.

  5. This is a <'h2'> tag. A <'H2'> tag marks the first sub-heading after your document's main heading. It defines the second-level headings on your webpage. Like a <'H1'> tag, a <'H2'> tag also appears larger than the rest of your main body text.

  6. This is a <'h3'> tag. The HTML <'h3'> tag defines the third level heading in the HTML document.

  7. This is a <'img'> tag. The <'img'> tag is used to incorporate in-line graphics (typically icons or small graphics) into an HTML document

  8. This is a <'a'> tag. The <'a'> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <'a'> element is the href attribute, which indicates the link's destination.

  9. This is a <'div'> tag. The <'div'> tag defines a division or a section in an HTML document. The <'div'> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The <'div'> tag is easily styled by using the class or id attribute.

  10. This is a <'p'> tag. The HTML <'p'> tag defines a paragraph. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

  11. This is a <'li'> tag. The <'li'> tag defines a list item. The <'li'> tag is used inside ordered lists(<'ol'>), unordered lists (<'ul'>), and in menu lists (<'menu'>).

  12. This is a <'ol'> tag. The <'ol'> tag defines an ordered list. An ordered list can be numerical or alphabetical.

  13. This is a <'ul'> tag. The <'ul'> tag defines an unordered (bulleted) list.

    14. Difference between class and ID

    Classes are like labels you can give to many things. They're good for styling lots of elements in a similar way. IDs are like special names for one thing only. They're best for styling one unique element. Classes are flexible for general styles, while IDs are strict and unique for specific styles.

    An example on how to assign class in html and CSS

    <'p class="my-class">This is a paragraph with a class.<'/p'> (html)

    .my-class { color: blue; font-size: 16px; font-weight: bold;} (css)

    An example on how to assign ID in html and CSS

    <'p id="my-id">This is a paragraph with an ID.<'/p'> (html)

    #my-id { color: red; font-size: 20px; font-weight: bold;} (css)
